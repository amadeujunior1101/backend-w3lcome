# API em NodeJs

Simples aplicação que manipula array de pessoas atraves dos metodos: index, store, update e delete

## Pacotes inclusos

1. Typescript
2. Express
3. Dotenv

Clonar do repositório:

    git clone git@github.com/amadeujunior1102/backend-w3lcome.git
    cd backend-w3lcome
    yarn

# Development
    yarn dev
